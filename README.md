## Projects

### 1. Blog Preview Card

**Description:**
A card layout showing a blog post preview with:

* Image
* Category label
* Published date
* Title and description
* Author info

**Folder structure:**

```
blog-preview-card/
├── index.html
├── style.css
├── assets/
│   ├── fonts/
│   └── images/
└── design/
```

**Picture**
![alt text](./blog-preview-card/preview.jpg)

---

**How to run:**

1. Open `index.html` in a browser.
2. All styling is included in `style.css`.
